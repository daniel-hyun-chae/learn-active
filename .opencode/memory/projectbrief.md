Project: Learn-Active (working title)
Summary:
Build a platform where people create and publish interactive learning courses made of short lessons and rich exercises, not primarily video.
Scope:
- Core experience: short explanations, interactive exercises, immediate feedback, and progress tracking.
- Core capability: a flexible exercise engine supporting multiple question types (multiple choice, fill-in-the-blank, reordering, matching, cloze).
- Authoring: course creators define lessons in structured formats; exercises are validated and graded automatically.
- AI assistance: AI can generate draft exercises, but humans review before publishing.
- Content model: structured, schema-based storage that scales beyond language learning.
Goals:
- Start with language learning (grammar-style exercises) while keeping the engine generic enough for math drills, logic problems, certification prep, coding quizzes, and other practice-based subjects.
- Prioritize scalable content structures and automatic validation/grading.
Non-goals:
- A video-first or passive learning platform.
