# Decision Log

This directory captures decisions about introducing new libraries or other implementation-affecting choices. When a new library is recommended for implementation, create a log entry here.

## File Naming
- Use the next sequential number and a short slug:
  - `decision-log/0001-<short-slug>.md`

## Template

```
# <Decision Title>

Date: YYYY-MM-DD
Status: Proposed | Accepted | Rejected | Superseded

## Context
<What problem are we solving and why now?>

## Decision
<What was decided and why?>

## Consequences
<Tradeoffs, risks, and follow-ups>

## Alternatives Considered
- <Alternative 1>
- <Alternative 2>

## Links
- <Relevant docs, issues, or evaluations>
```
